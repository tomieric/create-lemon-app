# create-lemon-app

Create an app inspired by [create-vite-app](https://github.com/vitejs/create-vite-app)


```
$ yarn create lemon-app <project-name>
$ cd <project-name>
$ yarn
$ yarn dev
```

## Preset

template list:

- [ ] vue(default)
- [ ] vite
- [ ] vite-component
- [ ] vue-component
- [ ] vue-h5
- [ ] vue-cli
- [ ] vitepress
- [ ] react
- [ ] react-h5
- [ ] preact
- [ ] taro
- [ ] uni-app
- [ ] flutter

To scaffold with specific template:

```
yarn create lemon-app --template vite
```
