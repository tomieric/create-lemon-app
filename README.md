# create-nike-app

Create an app inspired by create-vite-app,the `nike` just a name about my favorite brand.

```
$ yarn create nike-app <project-name>
$ cd <project-name>
$ yarn
$ yarn dev
```
## Tempaltes

Available templates:

- [ ] vue(default)
- [ ] vite
- [ ] vite-component
- [ ] vue-component
- [ ] vue-h5
- [ ] vue-cli
- [ ] vitepress
- [ ] react
- [ ] react-h5
- [ ] preact
- [ ] taro
- [ ] uni-app
- [ ] flutter

To scaffold with specific template:

```
yarn create nike-app --template vite
```
